# RailFlow Website

This repository contains the official landing page and web dashboard interface for **RailFlow** — an AI-powered Railway Traffic Control system.

## 🌟 Overview
The RailFlow Website serves as the front-facing portal for the project. It is designed with a modern, glassmorphism aesthetic (dark mode by default) and features:
- A responsive, animated hero section.
- Clear breakdowns of RailFlow's core metrics (Latency reduction, On-time performance improvements, and Manual intervention reduction).
- Direct links to the comprehensive **Deep Dive Documentation** and the underlying open-source simulation engine.

## 🛠️ Tech Stack
- **HTML5 & CSS3**: Pure, lightweight markup for ultra-fast load times.
- **Custom CSS Variables**: Theming engine utilizing `JetBrains Mono`, `Instrument Serif`, and `DM Sans` for premium typography.
- **No-JS Core**: The landing page is completely structural and styled without heavy JavaScript frameworks to ensure maximum performance and SEO score.

## 🚀 Deployment
This website is designed to be easily hosted via GitHub Pages, Vercel, or any static file host. Simply serve the `index.html` file at the root.
